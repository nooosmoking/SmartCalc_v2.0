Smart_Calc v2.0.

General information

The program inplement an engineering version of calculator and have a simpe interface with input windows and 'equal' button.
The engine of program works by Dijkstra's algorithm and every expresson translate into Reverse Polish (bracketless) notation and then calculate.
Also Smart_Calc builds graphs in new window with coordinate grid.

Operations

Smart_Calc has ordinary operations with numbers, like *, /, +, -, mod, pow.
User can prioritize operations by using (, ).
For delete last action simbol Backspace.

Functions

Smart_Calc has opportunity to calculate some functions, like cos(x), sin(x), tan(x), acos(x), asin(x), atan(x), sqrt(x), ln(x), log(x).
User can get a result with any number or can input unknown variable (x) and choose value x via input window.

Graph

Smart_Calc has opportunity to draw a graph of input expression automaticly when 'equal' button is pressed.
User can set up range of X and Y in input windows. In window user can see graph of a given expression.
